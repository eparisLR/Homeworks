# Installation

Create a virtualenv by using this command :__
``
python3 -m venv venv
``

Then install the dependencies using :__
``
pip install -r requirements.txt
``

## Or with docker

If you have Docker compose installed you can run this command : 

``
sudo docker-compose up
``
